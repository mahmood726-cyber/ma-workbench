## E156 Abstract (CURRENT BODY)

Can a central navigation portal improve discoverability across a large portfolio of browser-based evidence synthesis tools that would otherwise require users to know individual application URLs? We built a minimal hub page serving as the entry point for the complete evidence synthesis tool portfolio, providing categorised links, brief descriptions, and status indicators for each available application. The hub organises tools into functional categories including pairwise meta-analysis, network meta-analysis, diagnostic test accuracy, individual participant data, and utility tools with visual indicators showing maturity status. All portfolio links were validated as reachable with correct title matching across the full tool collection. Category-based organisation reduced the median number of clicks required to reach a target tool from direct URL entry, as measured by navigation path analysis across representative user task scenarios. A central hub could serve as a persistent entry point for reviewers and students discovering the tool ecosystem for the first time. The hub provides navigation only and does not aggregate data or results across the linked applications.
