WebR Validator — browser vs R (metafor)

Validate the pooled estimate, τ², I², and Q from the browser against R's metafor::rma. Two paths: (1) run WebR directly in this tab (~30 MB one-time download) or (2) export an R script, run in your local R, paste the output back. Either path produces a signed TruthCert-style receipt.
